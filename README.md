# Code4Kidz 🚀

**A free, open-source coding platform that teaches children real HTML & CSS through live, visual learning.**

![Version](https://img.shields.io/badge/version-0.3-purple)
![License](https://img.shields.io/badge/license-MIT-blue)
![Age Range](https://img.shields.io/badge/age-7--10-green)

---

## 🎯 Mission

Code4Kidz brings quality coding education to children in under-resourced communities. No downloads, no accounts, no expensive hardware—just open a browser and start building.

We believe every child deserves the chance to create with technology, not just consume it.

---

## 💡 The Problem

Children in low-income neighborhoods face barriers to coding education:
- Existing platforms require high-end devices or heavy downloads
- Block-based tools don't teach real code
- Most platforms are designed for older learners
- Teachers need extensive training to use them

**Code4Kidz removes these barriers.**

---

## ✨ Features

### Live Split-Screen IDE
- **Left panel:** Code editor with syntax highlighting
- **Right panel:** Live preview that updates every keystroke
- **No "Run" button needed** — see your changes instantly

### Guided Lessons
- Step-by-step instructions with visual goals
- AI mascot guides kids through each concept
- **Fail-forward challenges:** Learn by debugging intentional bugs
- Positive feedback and micro-animations

### Zero Friction
- Works on 8-year-old computers
- No installation, no login, no setup
- Runs entirely in the browser
- Works offline (after first load)

---

## 📚 Curriculum (MVP)

**Lesson 1:** HTML Basics  
Build your first webpage with headings and paragraphs

**Lesson 2:** CSS Fundamentals  
Add colors, fonts, and backgrounds to shapes and animals

**Lesson 3:** The Box Model  
Learn margins, padding, and borders by styling cards

**Lesson 4:** Creative Project  
Build your own mini webpage from scratch

## 🛠️ Technical Stack

**Pure vanilla web technologies:**
- HTML5
- CSS3
- Vanilla JavaScript (no frameworks)
- CodeMirror for syntax highlighting
- Live preview via `<iframe>` with `srcdoc`

**Why vanilla?**
- Zero install overhead
- Works on any browser
- Easy for volunteers to run locally
- No dependency rot, no build complexity
- Accessible to beginner contributors

**Hosting:** Static file hosting (GitHub Pages, Netlify, Vercel)

## 👥 Target Users

**Primary:** Children ages 7-10 with no coding experience  
**Secondary:** Community center staff, teachers, volunteers running group classes

---

## 🎨 Design Philosophy

- **Fail-forward learning:** Mistakes are part of the process
- **Visual feedback:** See the impact of every change
- **Accessible:** Large fonts, high contrast, plain language
- **Energetic:** Futuristic purple/neon aesthetic that feels alive
- **No gatekeeping:** Zero barriers to entry

---

## 📊 Success Metrics

- 70%+ completion rate for Lesson 1 (no adult help needed)
- Runs on 8-year-old hardware
- Facilitators rate confidence ≥4/5
- Kids say "I made something" in exit interviews
